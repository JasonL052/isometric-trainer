# Iso Quad Trainer

A lightweight PWA for tracking isometric quad exercises on iPhone.

**Live app:** https://jasonl052.github.io/isometric-trainer/

## Default Workout
- 5 sets
- 45 second holds
- 2 minute rest between sets
- Single leg mode (left then right each set)

## Features
- Single leg or both legs mode
- Customizable sets, hold time, and rest time
- Manual start between legs and after rest
- Settings adjustable during rest
- Vibration feedback at key moments
- Screen stay-awake during workout
- Recent session history

## Install
Open the live link in Safari on iPhone → Share → Add to Home Screen
